Общее Управление:

цифры(1, 2) - Смена градиента
WASD - Премещение
стрелка(вверх, вниз), СКМ - Зум
стрелка(вправо, влево) - Изменение количества итераций


Доп Управление для Множества Жулеа:

Изменение Множества:
    ЛКМ + СКМ - изменение реального параметра
    ПКМ + СКМ - изменение мнимого параметра

Shift(удерживать) - изменяет шаг для изменемия множества
